# Portfolio Web (Vite + React + TypeScript + Tailwind)

This is a scaffolded starter for a personal portfolio built with Vite, React (TypeScript), and Tailwind CSS. It includes a small `ui` folder with shadcn-style components (Tailwind-based) so you can start building immediately.

What's included
- Vite + React + TypeScript
- Tailwind CSS configured
- Basic `src` layout with a small `ui` component set
- ESLint + Prettier config (skeleton)

Next steps (run locally)

1. Install dependencies

```bash
npm install
```

2. Start the dev server

```bash
npm run dev
```

Firebase and GitHub setup will be added after you verify the local scaffold.
